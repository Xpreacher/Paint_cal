# Paint Calculator

Quick paint estimator by xpreacher. Enter your wall dimensions and get the number of paint cans needed. Assumes 1 can covers 5m².

### Features
- **Simple input**: Just enter height + width in meters
- **Instant calculation**: Tells you cans needed right away
- **Fixed coverage**: 5m² per can - standard paint coverage
- **No dependencies**: Pure Python
